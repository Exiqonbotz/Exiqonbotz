# 🔥 About Phoenix

Phoenix is an advanced **WhatsApp multi-bot platform** built with Node.js.

The system allows users to run **multiple WhatsApp bots simultaneously**,  
manage them via a **web dashboard**, and even **self-host their own instances**.

Main features:

• Multi-bot architecture  
• Selfhosting system  
• Web dashboard for bot control  
• Modular command system  
• German & English support  
• Economy, Games, Moderation and more

---

# 🧩 Phoenix Ecosystem

Phoenix is more than a single bot.
